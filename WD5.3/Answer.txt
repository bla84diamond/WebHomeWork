1.	Как исправить данную строчку, чтобы выключка текста была влево <ul class="navbar-nav mr-auto">?

Чтобы выключка текста была влево от навигационного меню, необходимо добавить класс `ml-auto` к `<ul>` элементу:

<ul class="navbar-nav ml-auto">
  <!-- пункты меню -->
</ul>

Класс `ml-auto` устанавливает отступ слева для элемента, равный ширине оставшейся свободной области. Таким образом, все элементы меню будут выравнены по левому краю, а выключка текста будет расположена справа.

2.	Что позволяет сделать утилита Spacing?

Утилита Spacing в Bootstrap позволяет быстро и удобно задавать отступы (margin и padding) для элементов на странице. Она предоставляет набор классов, которые можно применять к элементам HTML и которые задают нужные отступы.

Основные преимущества утилиты Spacing:

1. Простота использования. Для задания отступов не нужно писать CSS-код или использовать отдельный файл стилей. Достаточно просто добавить нужный класс к элементу.

2. Гибкость. Утилита Spacing позволяет задавать отступы как для всех сторон элемента (верхней, правой, нижней и левой), так и для отдельных сторон.

3. Адаптивность. Утилита Spacing позволяет задавать отступы для разных размеров экранов. Для этого используются префиксы классов, например, `mt-sm-4` задает отступ сверху для устройств с шириной экрана больше `576px`.

4. Возможность комбинирования классов. Утилита Spacing позволяет комбинировать классы для задания сложных отступов. Например, `mt-3 mx-lg-5` задает отступ сверху и снизу на 3 единицы и отступ слева и справа на 5 единиц для устройств с шириной экрана больше `992px`.

Таким образом, утилита Spacing в Bootstrap позволяет быстро и удобно задавать отступы для элементов на странице, обеспечивая гибкость, адаптивность и возможность комбинирования классов.
