1.	С какой целью создавался React?

React – это библиотека JavaScript, которая была создана с целью упрощения и оптимизации процесса создания пользовательских интерфейсов для веб-приложений. 

Основной задачей React является обеспечение быстрого и эффективного рендеринга пользовательских интерфейсов. React использует виртуальный DOM (Document Object Model), который позволяет избежать ненужных операций обновления интерфейса и ускорить процесс рендеринга.

React также предоставляет разработчикам удобный и понятный способ создания компонентов пользовательского интерфейса, которые могут быть использованы повторно в различных частях приложения.

Благодаря своей эффективности и удобству использования, React стал одним из самых популярных инструментов для разработки веб-приложений, используемых многими крупными компаниями, такими как Facebook, Instagram, Netflix, Airbnb и другие.
