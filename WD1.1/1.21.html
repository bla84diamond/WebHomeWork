Пример списка определений

HTML
Язык разметки гипертекста, используемый для создания веб-страниц.
CSS
Каскадные таблицы стилей, используемые для определения внешнего вида веб-страниц.
JavaScript
Язык программирования, используемый для создания интерактивных элементов на веб-страницах.
PHP
Скриптовый язык программирования, используемый для создания динамических веб-страниц и веб-приложений.
MySQL
Реляционная система управления базами данных, используемая для хранения и управления данными на веб-серверах.
